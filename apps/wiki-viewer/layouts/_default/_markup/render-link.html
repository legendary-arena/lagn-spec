{{- /* why: D-13809 default — out-of-tree links (`../**` from a wiki page) rewrite to GitHub blob URLs at build time. The wiki source paths are repo-root-relative when resolved from wiki/, so `../docs/ai/ARCHITECTURE.md` → `docs/ai/ARCHITECTURE.md` and `../packages/foo/bar.ts` → `packages/foo/bar.ts`.
